# Домашнє завдання до тем 7 та 8

## Завдання

- `task_01.py` — пошук найменшого значення у двійковому дереві пошуку.
- `task_02.py` — пошук суми всіх значень у двійковому дереві пошуку.
- `task_03.py` — мінімальна вартість з’єднання кабелів за допомогою купи.

## Пояснення рішень

### Завдання 1

Для пошуку найменшого значення використано двійкове дерево пошуку.

У двійковому дереві пошуку всі менші значення знаходяться в лівому піддереві, тому найменше значення можна знайти, рухаючись від кореня дерева вліво, поки існує лівий нащадок.

Тому немає потреби обходити все дерево.

### Завдання 2

Для знаходження суми всіх значень використано рекурсивний обхід дерева.

На відміну від пошуку мінімального значення, тут потрібно врахувати кожен вузол дерева. Тому функція рекурсивно додає значення поточного вузла, суму лівого піддерева та суму правого піддерева.

### Завдання 3

Для мінімізації вартості з’єднання кабелів використано мінімальну купу.

На кожному кроці потрібно об’єднувати два найкоротші кабелі, тому що це дозволяє зменшити загальну суму майбутніх об’єднань. Мінімальна купа дає змогу ефективно отримувати два найменші елементи.

## Примітка

Кожне завдання виконано в окремому Python-файлі.

У кожному файлі є приклад виконання та базові перевірки через `assert`.
